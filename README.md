<h1 align="center">MDControl</h1>

## Multi-Device Control

**MDControl** 是一个软件 KVM 工具，允许用户连接多台 Windows 设备，让键鼠在其之间**自由移动、自由传输剪贴板和文件**。

**MDControl** 拥有蓝牙控制和TCP控制功能，在无网络环境下也可正常使用；以及 UDP 广播和蓝牙扫描功能，能够快速连接活跃的设备。
