<h1 align="center">MDControl</h1>

### Multi-Device Control

---

> **MDControl** 是一个软件 KVM 工具，允许用户只使用一套键鼠在多台设备之间自由移动，以及自由传输剪贴板和文件。
> **MDControl** 拥有蓝牙控制和TCP控制功能，在无网络环境下也可正常使用。
